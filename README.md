# Музыкальный плейлист

Программа для скачивания текста и обложек песен

## Как установить

Далее создайте .env файл в папке с проектом. В нем создайте переменную `API_KEY`. В нем будет храниться ключ сайта GENIUS. Для получения этого ключа вам понадобится зарегистрироваться на сайте [GENIUS](https://docs.genius.com/)
```
API_KEY=ваш ключ
```

Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Как запустить

Чтобы запустить проект запустите командную строку, перейдите в папку с проектом и впишите одну из 3 строк приведенных ниже для запуска функции:

Функция для поиска артиста по его id. Чтобы указать id нужного артиста, найдите id на сайте GENIUS
```
python artists.py --id=id артиста
```

Функция для поиска песни по его названии
```
python artists.py --name=название песни
```

Функция для поиска песни по ее id. Чтобы указать id нужной песни, найдите id на сайте GENIUS
```
python artists.py --id=id песни
```

После запуска этих функций текст песен, а так же их обложки, будут скачены в созданную программой папку
